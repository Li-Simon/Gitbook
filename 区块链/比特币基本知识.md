# 地址

钱包由私钥与公钥（地址）组成，通过私钥可以得到公钥，这是由密码学保证的，类似于

$$Y = C^X%d$$ mod D  计算都是模掉一个大数D

其中C是已知的大的质数，X就是私钥，Y就是公钥，通过私钥X得到公钥Y很简单，但是通过公钥Y求解私钥X则很难

```
Key privateKey = new Key(); // generate a random private key
```
