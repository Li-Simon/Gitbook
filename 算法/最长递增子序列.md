# 最长递增子序列

对于序列5，3，4，8，6，7，其最长的递增子序列是3, 4, 6, 7,这里，不需要保证元素是连在一起的，只需要序号上升即可。问题变成求解n元系列a\[1\],a\[2\],a\[3\],...,a\[n\],对于i&lt;j&lt;n, 假设我们已经求得前i个元素的最长递增子序列长度为MaxLen\[i\] 满足i&lt;j，那么我们可以求得MaxLen\[j\]

```cpp
MaxLen = 0;
MaxLen[j] = 1;
for(i = 1; i < j; i++)
{
    if(a[j] > a[i])
    {
        MaxLen = MaxLen[i] + 1;
        MaxLen[j] = MaxLen[j] - MaxLen? MaxLen[j]:MaxLen;
    }
}
```
