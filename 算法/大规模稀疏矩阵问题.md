###Preconditioning
思考线性方程系统：$$\mathbf{Ax}=\mathbf{b}$$  
假设$$M=M_1M_2$$是非奇异的，思考线性系统$$\hat A \hat x = \hat b$$,其中：  
$$\kern{8 em} \hat X = M_1^{-1}A M_2^{-1}$$  
$$\kern{8 em} \hat b = M_1^{-1}b$$  
如果M与A很接近，因此$$\hat A$$接近于单位矩阵I。 因此可以通过$$\mathbf{M_2x} = \hat x$$.
